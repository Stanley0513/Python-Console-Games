# Python Console Games Collection 🎮

這是我在大一「計算機概論」課程中獨立開發的 Python 終端機互動遊戲合集。這些專案未使用任何外部圖形化套件，純粹透過 Python 內建語法實作，展現了基礎的演算法邏輯、一維/二維陣列處理以及終端機互動設計。

## 📂 收錄遊戲與功能亮點

### 1. 踩地雷 (Minesweeper) - `Minesweeper.py`
* **簡介：** 實作 9x9 經典踩地雷，包含 10 顆隨機生成的地雷。
* **功能：**
  * 支援座標與指令字串解析 (如輸入 `a5` 踩格，輸入 `a5f` 插旗標記)。
  * 利用**遞迴演算法 (Recursion)** 實作「踩到空白格時自動連鎖展開相鄰安全區域」的進階功能。
  * 內建時間戳記 (Time module) 計算遊玩總時長。

### 2. 21點撲克牌 (Blackjack) - `Blackjack.py`
* **簡介：** 玩家與莊家 (電腦) 對戰的 21 點遊戲。
* **功能：**
  * 實作完整的洗牌與發牌邏輯。
  * 具備動態數值判定條件，精準處理「ACE 牌」可視為 1 點或 11 點的切換機制。
  * 寫入莊家 AI 規則 (低於 17 點強制補牌)。

### 3. 四子棋 (Connect Four) - `Connect Four.py`
* **簡介：** 支援雙人對戰 (Player X vs Player O) 的重力四子棋。
* **功能：**
  * 利用一維陣列動態模擬 7x6 二維棋盤的視覺化輸出。
  * 實作「重力掉落」邏輯，確保玩家選擇行數後，棋子會自動落在該直排的最底層。
  * 具備嚴謹的勝負判定演算法，每次落子後即時掃描橫向、縱向與所有斜向的連線狀態。

## 🛠 執行環境與方式
* **開發語言：** Python 3
* **遊玩方式：** 開啟終端機 (Terminal / Command Prompt)，進入檔案所在目錄，輸入執行指令即可遊玩。
